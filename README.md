## Prerequisites

You will need the following things properly installed on your computer.

* Git
* Node.js (with NPM)


## Installation

* `git clone git@github.com:addmitriev/dins-test.git` this repository
* `cd dins-react`
* `npm install`

## Running / Development

* `npm start`
* Visit your app at http://localhost:3001.

## Running Tests

* `npm test`
* `npm run test:lint`
